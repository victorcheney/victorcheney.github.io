yum 安装最新版本nodejs

2019-11-24

示例是安装 10.x 版

第一步：

curl --silent --location https://rpm.nodesource.com/setup_10.x | sudo bash -

第二步：

sudo yum -y install nodejs

wget https://nodejs.org/dist/v10.17.0/node-v10.17.0-linux-x64.tar.xz
